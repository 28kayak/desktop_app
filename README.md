



# Install Electron
```sh
 npm install --save-dev electron
 npm install --save-dev nodemon
```

# Run Application
```sh
#run a application
npm start 

#run app with console output
npm run watch
```

# Debugging node.js side

#start app (port number is arbitrary )
electron --inspect=5858
